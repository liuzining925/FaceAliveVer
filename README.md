# 基于seetaface6人脸识别（活体检测）的封装

# 使用

## Step 1. Add it in your root build.gradle at the end of repositories:

```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

## Step 2. Add the dependency

```
dependencies {
        implementation 'com.github.zining925:FaceAliveVer:V1.0.0'
}
```
